# rust-study
러스트 스터디 내용을 기록하기 위한 저장소입니다.

##기본적인 룰
- 해당 스터디 범위 챕터를 다 읽고 실습해보고 오기 발표는 당일에 랜덤
- 범위를 정해서 스터디 전까지 다들 읽고 실습해오기
  - 스터디 당일날 랜덤 인원 선정
  - 이번 챕터 발표자
  - 전주 챕터 과제 발표자
  - 서기
- 스터디 진행 후 다음 스터디 범위는 그때 결정
- 끝난 시점에 간단히 회고

## 제약 조건
- 당일 날 갑작스레 불참하면 벌금(만원)
- 전날까지 양해를 구하면 괜찮은걸로
